Greetings!

在這個特別的一年，邁入春天之際時，在Zentangle的我們就體會到今年將會是比以往任何時候都更合適來創作。

儘管我們好像才剛完成Zentangle Project Pack No.10，但下一次的Zentangle Project Pack我們也已經準備好了。我們希望在新品發表和視頻發布之間預留足夠的時間，讓即便在遠方的朋友也能及時收到Project Pack。

同時也提醒您，所有的Zentangle Project Pack視頻都可以按照自己的時間規劃和節奏觀看。所以請用你最舒服的坐姿，放鬆心情，跟著我們一同享受Zentangle Project Pack系列。各位喜愛禪繞 Zentangle的朋友，我們向您介紹：
Zentangle Project Pack No. 11

在Zentangle Project Pack No.10系列裡，我們發現在禪繞的發展歷程中，潛藏著很多小故事。禪繞Zentangle®的創辦人Rick Roberts 和 Maria Thomas透過創作、教學和旅行，分享過很多類似的故事。

其中有一個大家比較熟知的就是Rick和Maria同樣地著迷於和斐波那契數列有關的幾何學、神聖比例和黃金比例；他們持續研究並發現這些存在於大自然、藝術、建築，甚至是行為中的圖案是如何與幾何學產生共鳴，這成了旅途中十分重要的部分

邀請大家加入我們的行列，一同繼續探索禪繞之旅。

別緊張，如同向日葵和鸚鵡螺不需要依賴計算器成長，Project Pack所帶來的系列單元也不需要任何精細的數學規劃或技巧。我們將在一個圖樣能夠成長茁壯的新框架中探索，也會深入探討網絡與片段的概念，透過精巧的陰影技法，無論是搭配黑色或彩色，都能享受在創作的樂趣中。

然後，讓我們找到一個自己最舒服的坐姿，懷著感恩與感謝的心來欣賞這一切。

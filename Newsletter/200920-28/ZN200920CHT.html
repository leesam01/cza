歡迎來到Zentangle Project Pack No.11的第一天！

在Zentangle Project Pack No.10系列裡，我們發現在禪繞的發展歷程中，潛藏著很多小故事。禪繞Zentangle®的創辦人Rick Roberts 和 Maria Thomas透過創作、教學和旅行，分享過很多類似的故事。 其中有一個大家比較熟知的故事就是Rick和Maria都著迷於和斐波那契數列有關的幾何學、神聖比例和黃金比例；去探索、學習這些存在於大自然、藝術、建築，甚至是各種圖案中的驚喜。 不知不覺中，探索這些幾何比例與序列已成為他們旅程中很大的一部分。

邀請大家加入我們的行列，一同繼續探索禪繞之旅。

就如同向日葵和鸚鵡螺不需要依賴計算器成長，Project Pack所帶來的一系列單元也不需要任何精細的數學規劃或技巧。所以請大家放輕鬆，我們將深入探討網絡與片段的概念，透過精巧的陰影技法，以及黑色與不同色彩的搭配。就讓我們一同來能享受以及欣賞這一切吧!

在介紹視頻中，Rick和Maria會介紹這次Project Pack裡的材料以及一個好玩的工具(Phi) Markus Operadus II。還沒有拿到Zentangle Project Pack No.11的朋友，您暫時先用手邊有的材料來練習。您可以在這裡點擊右側藍字先下載Zentangle Project Pack No.11的Markus Operatus II。

接著就讓我們一起跟著Molly與Martha引導，開始Project Pack No.11的創作。

Enjoy!

欢迎来到Zentangle Project Pack No.11的第一天！

在Zentangle Project Pack No.10系列里，我们发现在禅绕的发展历程中，潜藏着很多小故事。禅绕Zentangle®的创办人Rick Roberts 和 Maria Thomas透过创作、教学和旅行，分享过很多类似的故事。 其中有一个大家比较熟知的故事就是Rick和Maria都着迷于和斐波那契数列有关的几何学、神圣比例和黄金比例；去探索、学习这些存在于大自然、艺术、建筑，甚至是各种图案中的惊喜。 不知不觉中，探索这些几何比例与序列已成为他们旅程中很大的一部分。

邀请大家加入我们的行列，一同继续探索禅绕之旅。

就如同向日葵和鹦鹉螺不需要依赖计算器成长，Project Pack所带来的一系列单元也不需要任何精细的数学规划或技巧。所以请大家放轻松，我们将深入探讨网络与片段的概念，透过精巧的阴影技法，以及黑色与不同色彩的搭配。就让我们一同来能享受以及欣赏这一切吧!

在介绍视频中，Rick和Maria会介绍这次Project Pack里的材料以及一个好玩的工具(Phi) Markus Operadus II。还没有拿到Zentangle Project Pack No.11的朋友，您暂时先用手边有的材料来练习。您可以在这里点击右侧蓝字先下载Zentangle Project Pack No.11的Markus Operatus II。

接着就让我们一起跟着Molly与Martha引导，开始Project Pack No.11的创作。

Enjoy!

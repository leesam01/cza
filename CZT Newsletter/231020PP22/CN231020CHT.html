Hello CZT,

Bijou在Zentangle度過了一個非常忙碌的一年。簡單地說，他的創意思緒湧流不息，他已經準備好分享一些他對禪繞產品和內容的想法。他非常渴望為我們的產品開發團隊做出貢獻，他的堅持難以忽視，因此我們決定試試他的一些點子。

我們喜歡其中一個特別的靈感，並決定將它成為Zentangle Project Pack No. 22的完美主題，這也是我們2023年「12天禪繞畫創作系列」活動。我們將推出連續12天的禪繞教程、精選的材料，以及圍繞著相同創作主題的節慶內容。

正如許多禪繞愛好者所知——禪繞畫方法是一種容易學習、放鬆且有趣的方法，透過我們稱之為圖樣的結構性圖案，來創造美麗的圖像，而你可以透過點、線條、弧線、曲線和球體的組合來畫圖樣。在禪繞畫方法的歷史中，有一些圖樣比其它的圖樣更為人熟知、更經常使用，也更容易變化。它們的形式基本、很有彈性、熟悉，並且在許多方面能充分發揮潛力。

Zentangle Project Pack No. 22將深入探討一個特定的圖樣以及它所代表的一切。一如往常，我們採取非常跳脫框架的方式來玩這個有趣的主題，也同樣地希望在我們的影片發布之前不要透露太多細節。可以這麼說，這次的創作系列將深入研究我們最經典的其中一個圖樣，它將會仔細觀察一些啟發性的線條和潛在的馬賽克組合方式，也許……也許會有一個新的探索材料。

Zentangle Project Pack No.22 - Zentangle 2023年度12天主题的影片將在美東時間 2023年12月01日起透過Zentangle newsletter、官網與YouTube頻道發布。

還請記得，Zentangle Project Pack的內容元件一直是系列主題驚喜的一部份。我們呼籲已拿到Zentangle Project Pack的朋友，在Zentangle發布影片之前，不要共享Project Pack相關內容的照片。

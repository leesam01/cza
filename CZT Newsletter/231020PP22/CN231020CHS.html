Hello CZT,

Bijou在Zentangle度过了一个非常忙碌的一年。简单地说，他的创意思绪涌流不息，他已经准备好分享一些他对禅绕产品和内容的想法。他非常渴望为我们的产品开发团队做出贡献，他的坚持难以忽视，因此我们决定试试他的一些点子。

我们喜欢其中一个特别的灵感，并决定将它成为Zentangle Project Pack No. 22的完美主题，这也是我们2023年“12天禅绕画创作系列”活动。我们将推出连续12天的禅绕教程、精选的材料，以及围绕着相同创作主题的节庆内容。

正如许多禅绕爱好者所知——禅绕画方法是一种容易学习、放松且有趣的方法，透过我们称之为图样的结构性图案，来创造美丽的图像，而你可以透过点、线条、弧线、曲线和球体的组合来画图样。在禅绕画方法的历史中，有一些图样比其它的图样更为人熟知、更经常使用，也更容易变化。它们的形式基本、很有弹性、熟悉，并且在许多方面能充分发挥潜力。

Zentangle Project Pack No. 22将深入探讨一个特定的图样以及它所代表的一切。一如往常，我们采取非常跳脱框架的方式来玩这个有趣的主题，也同样地希望在我们的影片发布之前不要透露太多细节。可以这么说，这次的创作系列将深入研究我们最经典的其中一个图样，它将会仔细观察一些启发性的线条和潜在的马赛克组合方式，也许……也许会有一个新的探索材料。

Zentangle Project Pack No.22 - Zentangle 2023年度12天主题的影片将在美东时间 2023年12月01日起透过Zentangle newsletter、官网与YouTube频道发布。

还请记得，Zentangle Project Pack的内容元件一直是系列主题惊喜的一部份。我们呼吁已拿到Zentangle Project Pack的朋友，在Zentangle发布影片之前，不要共享Project Pack相关内容的照片。

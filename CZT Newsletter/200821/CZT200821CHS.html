Hello CZT,

在这个最有趣的一年，我们来到了春季，在Zentangle的我们体会到今年将会是比以往任何时候都更合适来创作。我们承诺会为遍布全球的Zentangle社群提供更多的创作发想。

尽管好像才刚完成了Zentangle Project Pack No.10，但我们确实已经为下一次的Zentangle Project Pack准备就绪。我们希望在新品发表和视频发布之间预留足够的时间，让即便在远方的朋友也能及时收到Project Pack。

这是我们为CZTs所公布的预告。我们将于2020年9月20日发布Zentangle Project Pack No.11。

同时也提醒您，所有的Zentangle Project Pack视频都可以按照自己的时间规划和节奏观看。所以请用你最舒服的坐姿，放松心情，跟着我们一同享受Zentangle Project Pack系列。各位喜爱禅绕 Zentangle的朋友，我们向您介绍：
Zentangle Project Pack No. 11

在Zentangle Project Pack No.10系列里，我们发现在禅绕的发展历程中，潜藏着很多小故事。禅绕Zentangle®的创办人Rick Roberts 和 Maria Thomas透过创作、教学和旅行，分享过很多类似的故事。

其中有一个大家比较熟知的就是Rick和Maria同样地着迷于和斐波那契数列有关的几何学、神圣比例和黄金比例；他们持续研究并发现这些存在于大自然、艺术、建筑，甚至是行为中的图案是如何与几何学产生共鸣，这成了旅途中十分重要的部分。

邀请大家加入我们的行列，一同继续探索禅绕之旅。

别紧张，如同向日葵和鹦鹉螺不需要依赖计算器成长，Project Pack所带来的系列单元也不需要任何精细的数学规划或技巧。我们将在一个图样能够成长茁壮的新框架中探索，也会深入探讨网络与片段的概念，透过精巧的阴影技法，无论是搭配黑色或彩色，都能享受在创作的乐趣中。

然后，让我们找到一个自己最舒服的坐姿，怀着感恩与感谢的心来欣赏这一切。
